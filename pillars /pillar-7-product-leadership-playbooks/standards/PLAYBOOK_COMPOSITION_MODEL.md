# Playbook Composition Model

Defines how playbooks operate together.

---

# Core Principle

> Compose execution, not responsibilities

Composition must never result in:

- interpretation of signals  
- evaluation of outcomes  
- emergence of decision-making logic  

---

# Composition Types

## Sequential
Playbooks operate across the PLOS loop.

## Parallel
Playbooks operate independently in different systems.

## Cross-System
Playbooks coordinate across systems without merging roles.

---

# Rules

- No boundary collapse  
- No responsibility transfer  
- Preserve mediation path  
- Execution only  

---

# Anti-Patterns

- decision emergence from chaining  
- signal interpretation across playbooks  
- hybrid system behavior
