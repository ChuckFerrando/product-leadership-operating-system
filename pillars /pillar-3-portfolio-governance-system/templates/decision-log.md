# Portfolio Decision Log

This document records material portfolio and investment decisions made through the governance cadence.

The objective is transparency, traceability, and institutional memory.

All strategic, capital allocation, and portfolio tradeoff decisions must be documented.

---

# Logging Principles

1. Every decision must have a documented owner.
2. Tradeoffs must be captured explicitly.
3. Decisions should reference supporting artifacts (investment memos, scoring updates, risk reviews).
4. Decisions should be written clearly enough that a reader months later can understand the rationale.

---

# Decision Record Format

Each decision should capture:

| Field | Description |
|------|-------------|
| Decision ID | Unique identifier |
| Date | Decision date |
| Forum | Governance forum where decision occurred |
| Initiative / Topic | Initiative or portfolio topic |
| Decision | What was decided |
| Rationale | Why the decision was made |
| Tradeoffs | What alternatives were considered |
| Owner | Responsible executive |
| Follow-Up Actions | Required next steps |
