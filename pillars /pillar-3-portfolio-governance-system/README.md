# Portfolio Governance System

## Purpose

This repository documents **Pillar 3** of the **Product Leadership Operating System (PLOS)** and defines how organizations govern investment decisions through evaluation, prioritization, sequencing, funding, review, and rebalance across the portfolio.

---

## Diagram

![Pillar Diagram](../../assets/diagrams/pillar-3-portfolio-governance.png)

---

## System Role in PLOS

The **Portfolio Governance System** converts strategic direction into governed portfolio commitments.

It is responsible for:

- initiative intake
- evaluation and prioritization
- portfolio tradeoff decisions
- sequencing and funding decisions
- portfolio review and rebalance

---

## What This Pillar Contains

This pillar contains:

- canonical governance system definitions
- governance decision models
- portfolio review structures
- prioritization and sequencing models
- governance control artifacts
- system diagrams and supporting models

---

## Boundary Rules

Pillar 3 owns governance decisions. It does not own delivery execution, outcome evaluation, or measurement interpretation.

It must not:

- execute delivery work
- interpret raw Decision Intelligence signals directly
- bypass Customer Outcomes when evaluating outcome meaning
- redefine strategy outside strategic authority
- absorb playbook or lab responsibilities

---

## Relationship to Other Pillars

- **Pillar 1** defines the structural place of governance in the architecture
- **Pillar 2** defines the operating model within which governance runs
- **Pillar 4** executes governed work
- **Pillar 5** provides evaluated outcomes and learning inputs
- **Pillar 6** provides measurement visibility, not decision logic
- **Pillar 7** provides reusable governance execution playbooks
- **Pillar 8** provides controlled experiments that may be reviewed for promotion
