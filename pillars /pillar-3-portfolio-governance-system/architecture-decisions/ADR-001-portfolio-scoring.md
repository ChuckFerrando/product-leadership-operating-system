# ADR-001: Portfolio Scoring Framework

## Status

Accepted

## Context

Technology organizations must prioritize initiatives competing for limited engineering capacity.

Without a consistent evaluation mechanism, portfolio decisions become inconsistent and difficult to justify.

## Decision

Adopt a portfolio scoring framework that evaluates initiatives using standardized criteria including:

- strategic alignment
- customer impact
- delivery complexity
- risk profile

The scoring framework informs capital allocation decisions and portfolio prioritization.

## Consequences

Positive:

- improved transparency in investment decisions
- consistent initiative evaluation
- traceable portfolio governance

Tradeoffs:

- requires leadership agreement on scoring criteria
- introduces governance overhead
